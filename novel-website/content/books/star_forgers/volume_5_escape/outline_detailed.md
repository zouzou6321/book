# 第五卷细分大纲 (Chapter 071-090)

## 第一阶段：黄昏 (071-080)
- [ ] 071. 留守者：2100年的地球，资源耗尽，环境崩溃，人类退回城邦时代。
- [ ] 072. 轨道废墟：曾经繁华的同步轨道环因内战和缺乏维护，变成巨大的太空垃圾场。
- [ ] 073. 信号重现：奥尔特云边缘的神秘信号再次出现，这次是在地球轨道。
- [ ] 074. 智子封锁：信号被证实是高级文明的探测器，锁死了地球的基础物理。
- [ ] 075. 长征号的回响：就在绝望之时，收到长征号发回的最后一条消息。
- [ ] 076. 比邻星的幽灵：长征号在比邻星发现早已灭绝的文明遗迹。
- [ ] 077. 大过滤器：揭示宇宙真相——所有文明最终都会自我毁灭或被毁灭。
- [ ] 078. 最后的火种：地球上的幸存者试图重启被废弃的月球投射器。
- [ ] 079. 逆向工程：利用长征号传回的数据，试图破解外星封锁。
- [ ] 080. 太阳熄灭：外星探测器开始干扰太阳活动，引发超级耀斑。

## 第二阶段：永恒 (081-090)
- [ ] 081. 逃亡计划B：地球已经无法拯救，唯一的希望是把更多的人送出去。
- [ ] 082. 重启天梯：修复已经断裂的太空电梯，这是最后的生命线。
- [ ] 083. 诺亚方舟二号：利用轨道废墟拼凑出第二艘恒星际飞船。
- [ ] 084. 筛选：谁能上船？残酷的抽签决定人类的命运。
- [ ] 085. 最后的晚餐：留在地球上的人们举行最后的狂欢。
- [ ] 086. 起航：方舟二号在太阳风暴爆发前一刻点火。
- [ ] 087. 毁灭：太阳耀斑吞没地球，地表文明终结。
- [ ] 088. 孤独的旅程：两艘飞船在茫茫宇宙中相隔光年，互道珍重。
- [ ] 089. 新家园：长征号抵达一颗宜居行星，开始播种。
- [ ] 090. 铸星者：人类不再是地球人，而是星际种族。全书完。
