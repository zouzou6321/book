# 第197章：最后的晚餐

**时间**：世界重置前夕（逻辑层面的星期三深夜 23:59:59）
**地点**：李疯狂的深层意识核心（崩溃数据流中）

意识并没有像预想中那样坠入绝对的虚无。

所谓的“死亡”，或者说“格式化”，实际上更像是一种极其粗暴的强制关机，紧接着是拔掉电源后的余震。

李疯狂感觉自己变成了一串二进制的幽灵，漂浮在一片漆黑、粘稠的数据海洋里。这里没有上下左右之分，只有无数个破碎的“时间帧”像深海发光水母一样从他身边游过。

他低头看了看自己。

那具被董事长啃食得支离破碎的肉体已经不见了。取而代之的，是一团由金色代码与暗红噪点纠缠而成的光团。这团光并不稳定，时不时就会像接触不良的灯泡一样闪烁一下，崩解出几片关于“苏浅”或者“炸鸡”的记忆碎片。

【警告：人格完整性跌至 0.04%。】
【正在清理垃圾文件……已拦截。】
【检测到顽固残留数据，文件名：`Su_Qian_Forever.lock`。】

冰冷的系统提示音不再是从左耳或者视网膜上响起，而是直接在他的“灵魂”里震动。

“别费劲了。”李疯狂的意识波动了一下，发出了类似嗤笑的杂音，“那个文件是只读属性，还是隐藏的系统关键级。你删不掉它，除非你也跟着死机。”

四周的数据流突然湍急起来。

黑暗中，出现了巨大的阴影。那是“旧日餐厅董事长”没有被完全消化的意志残留，像是一块巨大的、腐烂的牛排油渍，漂浮在这个新生的维度里。

这团油渍试图侵蚀李疯狂，想要将这个试图掀桌子的家伙彻底同化为餐厅的一部分。但李疯狂现在已经是“规则”的一部分了。他不再是食客，也不再是食材，他变成了“消化酶”。

“你也想尝尝被我‘篡改’的味道吗？”

李疯狂猛地膨胀，那枚倒“V”符号不再只是一个图形，而变成了一把锋利的剪刀。

咔嚓。

他张开那个由概念构成的巨口，一口咬住了董事长的残渣。

【吞噬目标：旧日支配者意志（过期版）】
【转化中……】
【获得材质：绝对的傲慢与贪婪】
【副作用：严重的消化不良】

味道很难受。像是在喝一万度高温的辣椒油，又像是在咀嚼冰冷的铁钉。那种高维度的恶意在他体内横冲直撞，试图撑爆他的逻辑容器。

但他没有停下来。

因为他看到了。

在他的“胃”里，也就是这片意识空间的最深处，摆放着一张巨大的圆桌。那是他为自己准备的“最后的晚餐”。

餐桌周围坐着无数个“李疯狂”。

有穿着格子衫、盯着电脑屏幕敲代码的社畜李；
有在精神病院里流着口水、拿着“男科医院”传单当宝物的阿傻李；
有浑身长满触手、在南极冰原上与神明肉搏的半神李；
还有那个刚刚撕碎纸条、满脸决绝地冲向深渊的李疯狂自己。

这不仅仅是一张餐桌，这是他这二十六年人生的所有切片。

“真是一群烂人啊。”李疯狂看着这些过去的自己，发出了由衷的感叹，“怂的怂，疯的疯，没一个像话的。”

但他伸出手，温柔地抚摸过每一个自己的头顶。

“但也正是这群烂人，拼凑出了现在的我。拼凑出了那个不让世界吃掉她的‘我’。”

他坐到了主座的位置上。

面前没有食物，只有一把巨大的、正在滴着逻辑墨水的红笔。

那是他作为“世界管理员（临时）”持有的最高权限笔。

“好了，我们来干活吧。”

李疯狂抓起那笔，对着虚空中的日历狠狠划去。

原本的世界逻辑是：古神通过每日的侵蚀，逐渐吞噬理智，直到世界变成疯人院。
旧日的规则是：星期一是开胃菜，星期二是主菜，星期三是……

“太慢了，也太没劲了。”

李疯狂手中的红笔爆发出璀璨到令人目盲的光芒。

“这种细水长流的吃法，不符合我‘急食症’的性格。”

他用力在日历的“星期三”和“星期五”之间，画下了一道深不见底的鸿沟。然后，他将所有的疯狂、所有的异常、所有不可名状的古神低语、所有人类的噩梦，一股脑地扫进了这道鸿沟里。

【输入指令：概念覆写】
【参数设定：七天侵蚀 -> 单点爆发】
【修正对象：星期四】

“听好了，你们这群住在高维度的寄生虫。”

李疯狂一边疯狂地书写着新的底层代码，一边对着虚空咆哮，声音里带着一种近乎癫狂的快意。

“既然你们想吃‘疯狂’，那我就给你们加餐！”

“我把这里设为一个独立的‘排污口’，一个巨大的‘垃圾回收站’。每周一天的二十四小时，整个世界的理智阀门会暂时关闭。”

“在这一天，所有的规则由我定义。你们可以尽情地狂欢，肆意地进食，把地球当成自助餐随便吃。”

“但是——”

李疯狂手中的笔尖猛地顿住，左眼的金色电路在这一刻燃烧到了极致。他利用体内最后的“人性”，也就是胸口那个叫做苏浅的Bug，在疯狂的代码里刻下了一道绝对不可逾越的防火墙。

“过了这一天，哪怕再多一秒钟，你们也得给我给我滚回去！”

“这里是我在买单。这顿疯狂盛宴的账单，由我来签！”

系统屏幕上的红色警报疯狂闪烁，然后在一瞬间全部变成了绿色的【OK】。

【逻辑重构中……99%……100%】
【新规则已上线：疯狂星期四（狂欢版）】
【代价：执行者（李疯狂）将不复存在，化作规则本身。】

巨大的吸力从四面八方传来。李疯狂感觉自己的意识正在被稀释，被拉扯成无数条细丝，然后编织进世界的每一缕风中，每一滴雨里，每一块炸鸡的酥皮上。

他要变成“星期四”本身。

他要成为那个在每周的第四天，替人类承受所有疯狂的容器。

这是最完美的牺牲，也是最疯癫的赌博。

在意识即将彻底消散的最后一刻，他突然想起了什么。

“等等，等等。”

他拼尽全力，在那庞大到足以淹没宇宙的新规则代码里，藏了一行极小、极小的乱码。

`if (day == Thursday && user == "Su Qian") { protect_level = MAX; happiness_coupon = true; }`

这是给苏浅的“超级VIP特权”。

在这疯狂的星期四，哪怕全世界都在尖叫，唯独她，能收到一张带着他指尖温度的优惠券。

“搞定。”

李疯狂——不，现在应该称之为“星期四这一概念的具象化”——轻轻放下了笔。

他感到一种前所未有的轻松。就像加完了一个通宵的班，终于把那个改了一百遍的需求给怼上线了。

周围的数据流开始崩塌，化作现实世界中的风雪。

南极的冰层深处。

苏浅跪在地上，双手死死抓着那片已经失去温度的虚空。她的哭声在这个死寂的世界里显得格外刺耳。

“……我还没吃完那顿饭呢。”她喃喃自语。

突然。

一股极其微弱，却温暖得让人想流泪的热度，从她手中的无名指传了过来。

那是一片淡淡的金色光晕。光晕慢慢凝聚，最后化作了一个歪歪扭扭的、像是小孩子随手画上去的印记。

那是一个倒着的“V”，被一个圆圈圈住，看起来像是一个笑脸，又像是一个……勺子。

紧接着，她的脑海里，并没有响起那种冰冷的各种警告，而是回荡起了一个熟悉的声音。

那个声音有些慵懒，有些沙哑，还带着一点刚睡醒的鼻音。它不是在说话，而是在这片被重置的世界里，作为一种“背景音”永恒地流淌着：

“……别哭。这顿饭，算我请你的。”

“……记得V我50。”

咔哒。

仿佛是谁关掉了黑暗的开关。

南极上空那原本狰狞的黑色云层，突然裂开了一道缝隙。一道苍白却不刺眼的晨光透了进来，照亮了苏浅满是泪痕的脸。

风停了。

那种压抑得让人窒息的神性威压，那种时刻笼罩在头顶的末日阴影，在一瞬间消失得无影无踪。

世界变得无比安静，安静得就像是一个刚被打扫干净的餐厅，正准备迎接第一批早起的客人。

苏浅愣愣地看着自己的手心。

那个印记还在，甚至随着她的心跳，微微发烫。

她知道他没骗她。

他确实没死。他只是……融入了这里，变成了这个世界的“肠胃”，变成了那个把疯狂消化掉、只把平静留给她的星期四。

“疯子。”

苏浅擦干了眼泪，慢慢地站了起来。她的眼神不再迷茫，那个曾经柔弱的护士已经不在了，站在那里的，是旧日餐厅唯一的“合伙人”，是这一世唯一的“食客”。

她从怀里掏出那支已经被冻得有些发硬的录音笔，按下播放键。

里面原本是一段乱码，但现在，在这被李疯狂修改过的新世界里，乱码自动解码，变成了一段清晰而欢快的音乐。

那是《两只老虎》的调子，但被那个疯子用电子合成音改成了奇怪的摇滚版。

苏浅握紧了录音笔，转身走向通往地面的电梯。

“等着我。”

她在心里默默说道。

“你请客归请客，但服务费，我可是要收很贵的。”

（本章完）
