# 人物档案 v3

## 核心人物

### 林远 (Lin Yuan)
- **初登场年龄**：34岁 (2027年)
- **身份**：航天结构工程师，天工一号首批驻站人员。后成为天工城元老。
- **背景**：哈工大焊接专业本科，航天五院结构设计所工程师。参与过天宫空间站和嫦娥探测器的结构设计。手上有真正的焊工茧。
- **性格**：沉默寡言，只关心手上的活。不做演讲，做焊接。但在关键技术讨论时，他说的每句话都有分量——因为每句话背后都有实操经验。
- **人物弧光**：基层工程师 → 太空制造的实操专家 → 天工城的技术元老 → 随远航号前往小行星带 → 在天工三号退休。他一辈子都在建造东西，最后他建造的东西变成了一座城市。

### 赵长河
- **初登场年龄**：60岁 (2025年)
- **身份**：天工计划总设计师。中国航天科技集团总工程师。
- **背景**：主持过载人航天、嫦娥探月等工程。航天系统内最有威望的技术官僚。
- **性格**：沉稳、务实、不讲废话。对上能扛住政治压力，对下能护住工程师。他的字典里没有"大概""差不多"这种词。
- **名言**："航天工程不接受形容词。给我数字。"
- **人物弧光**：从体制内总师 → 力排众议推动天工立项 → 指挥小行星捕获 → 退休后在近地轨道天工城安家。

### 苏晚 (Su Wan)
- **初登场年龄**：36岁 (2027年)
- **身份**：中科院金属研究所研究员。天工一号驻站材料学家。
- **专长**：微重力冶金、太阳能聚光熔炼、小行星矿石加工工艺。
- **性格**：严谨到近乎刻板，实验室里绝不允许"差不多"。但当她看到微重力下完美生长的晶体时，眼睛会亮起来——那是她唯一流露情感的时刻。
- **名言**："在地面上，重力是暴君。每一个原子都被迫服从它的秩序。在那上面，原子是自由的。"
- **人物弧光**：地面实验室研究员 → 太空驻站科学家 → 攻克太阳能熔炼小行星工艺 → 太空冶金体系的创建者。

### 陈默 (Chen Mo)
- **初登场年龄**：28岁 (2025年)
- **身份**：新华社科技记者，后成为天工计划的官方纪录片导演/编年史作者。
- **功能**：读者的眼睛。他代表普通人的好奇和敬畏。
- **性格**：热血、好奇、执着。他不懂火箭方程，但他能把火箭方程讲成让14亿人都听得懂的故事。
- **人物弧光**：记者 → 纪录片人 → 记录天工计划从立项到小行星带建城的全过程。

## 次要人物

### 林小萱 (Lin Xiaoxuan)
- 林远的女儿。在天工城出生成长。第一代太空人。
- 从小在太空长大，对地球没有乡愁，只有好奇。
- 成长为新一代太空工程师，成为天工三号的核心技术负责人。第二代太空人的代表。

### 方致远 (Fang Zhiyuan)
- 天工一号的航天医学官。负责驻站人员健康和生命保障系统。
- 沉着冷静，危机中永远是最稳的人。
