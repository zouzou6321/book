# 第50章：产品

二〇四七年四月。天工城五号制造舱。

一面直径五十米的抛物面天线正在成型。

从制造舱的观察窗看出去，它像一朵正在绽放的金属花。骨架是蛛网状的碳纤维桁架——碳纤维用BL₁₁小行星的碳质物烧制，桁架节点用镍铁合金铸件连接。反射面是厚度仅零点一毫米的铝箔，在微重力下以静电展开方式铺设到骨架上——每一片铝箔宽两米、长六米，由织工机器人一片一片精确地黏合到位。

总计四百一十二片铝箔。拼合后形成一个近乎完美的抛物面——面型精度优于均方根零点三毫米。这个精度在地球上需要精密机床逐点加工，在太空中只需要让铝箔在零重力下自然展开到骨架的预设曲面上——没有重力变形，没有风载扰动，没有热膨胀不均匀。

五十米直径。在地球上，这个尺寸的天线不可能整体制造和发射。最大的通信卫星天线——地面折叠后整流罩内展开——直径约二十二米，已经是工程极限。再大，运载火箭的整流罩塞不下。

天工城不需要整流罩。天线在太空中制造，在太空中展开，在太空中直接部署到目标轨道。从来不需要经历发射时的振动、加速度和气动载荷。

这意味着结构可以做到极致轻薄。五十米天线的总质量只有二点八吨——如果在地球上建造一个同等尺寸的天线，结构质量至少需要二十吨以上，因为必须承受自身重力和运输载荷。

二点八吨。如果用长征九号甲从地球发射一颗带有同等天线面积的通信卫星，光天线的运输成本就是九十八万美元——还不算卫星本体、发射服务和保险。天工城的制造成本——原材料来自小行星，能源来自聚光镜，劳动力已经在轨——边际成本接近零。

这不是科研产品。这是一个商业命题。

周德志在三月的工作会议上把这张账算清了。他在白板上画了一个表格：

| | 地球制造+发射 | 天工城制造+在轨部署 |
|---|---|---|
| 天线直径 | ≤22米 | ≤80米（现有能力） |
| 天线质量 | 15-25吨 | 2-4吨 |
| 运至GEO费用 | $500-800万 | ~$0（已在轨） |
| 制造周期 | 18-24个月 | 3-4个月 |
| 面型精度 | RMS 0.5mm | RMS 0.3mm |

"结论很简单，"周德志说。"对于大型天线这个品类——直径超过二十米的——天工城在成本、尺寸、精度和交付时间上全面优于地球制造。不是好一点，是好一个数量级。"

五月。第一面五十米天线交付。

客户是中国电信集团——他们的下一代高通量通信卫星需要一面大口径天线来覆盖亚太地区的Ka波段通信。地球上没有厂商能提供超过二十米的整体天线。天工城可以。

天线从五号制造舱移出，由两台大型拖曳机器人推送到转移轨道。目标：地球静止轨道，东经一百一十度上空。卫星本体仍然从地球发射——天工城还不能制造复杂的电子载荷和转发器——但天线在轨道上与卫星对接。

两个来自完全不同世界的组件在三万六千公里高度的轨道上合为一体。一个从地球的引力井中挣扎着爬上来，一个在太空中从一颗小行星上生长出来。

天线展开成功。Ka波段信号测试——增益比设计指标高零点三分贝。在轨性能完美。

这是天工城的第一笔商业订单。合同金额——苏晚听到这个数字时挑了一下眉——两亿四千万元人民币。不算多。但这是太空工业第一次向地球市场出售制成品。方向比金额重要。

六月。第二个产品线启动。

光学镜面。

天工城的微重力铸造工艺——苏晚发展了十九年的那套技术——有一个地球上任何工厂都无法复制的优势：完美的对称性。

在地球上铸造一面大型抛物面镜，熔融玻璃或金属在模具中凝固时，重力会导致一侧的液面比另一侧高。冷却收缩不均匀，镜面产生微米级的变形。必须经过长时间的研磨和抛光来修正。哈勃太空望远镜的主镜——直径二点四米——研磨了两年。

在微重力下，这些问题不存在。

熔融的玻璃或金属合金在旋转的模具中凝固——利用离心力形成抛物面。在微重力下，没有重力干扰，离心力是唯一的成形力。冷却后的镜面是一个数学意义上近乎完美的旋转抛物面。面型误差可以低于均方根十纳米——十亿分之十米。不需要研磨。

苏晚的团队在六月成功铸造了一面直径四点二米的铝铍合金抛物面镜。面型精度：RMS 八纳米。

地球上最好的光学研磨工艺——花两年时间——能做到的精度大约是RMS十五纳米。天工城用两天的铸造时间做到了八纳米。

这面镜子的客户是中国科学院国家天文台。它将成为下一代太空望远镜的主镜——直接部署在日地拉格朗日L2点。不需要从地球发射——一台离子推进拖船从天工城出发，把镜子送到一百五十万公里外的L2点，大约需要四个月。

合同金额：十一亿元人民币。

周德志在七月的报告中写了一段话：

"天工计划的经济逻辑正在改变。过去十九年，天工是一个纯投入项目——国家投资，回报是战略能力和技术积累。从二〇四七年起，天工开始产生直接经济回报。太空制造的产品不是'未来的可能性'，而是今天的订单。大型天线、光学镜面、特种合金构件——这些产品的共同特征是：在太空制造比从地球制造并发射更便宜、更好、更快。"

"这是一个转折点。太空工业不再需要国家补贴来证明自己存在的意义。它开始自己养活自己。"

九月。天工城收到了第三笔订单——来自欧空局。一面直径六点五米的远红外望远镜主镜。合同谈判中欧方代表问了一个问题："你们的产能如何？能同时承接多少订单？"

苏晚回答了这个问题。

"目前，我们有十条铸造产线，六面专用聚光镜提供热源，约一百二十名冶金和光学工程师。同时承接两到三个光学镜面项目和四到五个天线项目。"

她停了一下。

"但我们的产能不是固定的。我们仍然在捕获小行星。每一颗新的小行星都会增加原材料供应。每增加原材料，我们就建造更多聚光镜。更多聚光镜意味着更大的铸造产能。"

她看着屏幕上的欧方代表。

"我们的产能在指数增长。如果你们现在不下单，一年后你们的竞争对手会排在你们前面。"

欧空局在两周内签了合同。

陈默在年末的纪录片里用了一个长镜头：天工城外的太空中，一面五十米的银色天线和一面四米的金色镜片并排漂浮，等待转移拖船的到来。背景是地球——蓝色的弧形轮廓占据了画面的下方三分之一。

旁白很短："天工城的第一批产品。它们的原材料来自四十六亿年前的太阳系。它们的目的地是今天的人类文明。中间的环节是一群住在太空里的工程师，和他们造的一切。"

镜头缓缓拉远。天线和镜片越来越小，天工城的轮廓出现在画面边缘——那个巨大的旋转圆柱体，暗银色的金属表面在阳光下缓慢转动。它的旁边，几颗小行星安静地悬浮着，像是等待被开启的仓库。

远处，三条几乎看不见的淡蓝色光束——离子推进器的尾迹——正在拖着三颗新的小行星缓缓驶来。
