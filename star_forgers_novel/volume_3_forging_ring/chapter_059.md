# 第59章：盘古

二〇五四年二月十三日。飞行第六百七十八天。

远航号最后一次化学推进制动。

四台液氧甲烷发动机点火四分十二秒，速度增量负一点七公里每秒。飞船相对于盘古小行星的速度降至每秒三点一米——接近于零。从轨道力学的角度，远航号已经被盘古的引力——虽然极其微弱——"捕获"了。

赵阳在指挥席上宣布："远航号进入盘古环绕轨道。轨道高度约两公里。轨道周期约十四小时。"

两公里。距离一颗直径八百米的小行星两公里。

导航段的观察窗外——盘古小行星缓慢地充满了视野。

它不像想象中的那样。

在地球上，人们想象小行星时通常会想到两种形象：要么是好莱坞电影里光滑的球形巨石，要么是望远镜照片上模糊的亮点。盘古两者都不是。

它的形状不规则——大致呈椭球形，长轴约九百米，短轴约七百米。表面被撞击坑覆盖——大大小小几百个，最大的一个直径超过两百米，深约四十米，占据了小行星的整个"北半球"。撞击坑之间是嶙峋的山脊和沟壑，像是一张被揉皱又展开的锡箔纸。

颜色出乎意料。不是灰色——是暗银色，带着金属特有的冷峻光泽。在阳光直射的区域，表面反射出刺眼的高光——那是裸露的金属面，没有风化层覆盖。在阴影区域，表面呈深黑色，几乎完全吸光。明暗交界线极其锐利——没有大气散射，阳光和阴影之间没有过渡，像是用刀切开的。

盘古在自转。周期约六点三小时。从远航号的轨道上看，它像一座缓慢翻转的山——每隔几小时，新的表面从阴影中转出，暴露在阳光下。旋转轴倾斜约十五度，使得某些区域终年不见阳光——极区的永久阴影坑。

苏晚站在观察窗前。

她的右手按在窗框上——手指因为微重力下的血液重新分布而略微浮肿，这是二十六年太空生活的标志。她看着盘古的表面从窗外缓缓滑过。

"光谱仪对准那个大撞击坑的内壁。"她在通信频道里对导航段的光学工程师说。

十分钟后，数据回来了。

近红外反射光谱显示了清晰的铁-镍合金吸收特征——零点九微米和一点八微米处的双峰，与实验室中纯镍铁样品的光谱高度吻合。进一步分析显示：镍含量约百分之八点五，铁含量约百分之八十九。另有钴百分之零点六，磷百分之零点三，余量为微量碳和硫。

百分之八点五的镍。这个数字让苏晚微微吸了一口气。

天工城此前处理过的近地小行星——M型——镍含量在百分之五到百分之十二之间。百分之八点五处于中间值。这个成分比例意味着盘古的主要矿物是铁纹石（低镍，体心立方晶格）和镍纹石（高镍，面心立方晶格）的共生体——标准的铁陨石结构，和四十六亿年前太阳系形成时的原始金属凝聚体一致。

她知道怎么冶炼这个。她已经炼了二十五年。

更让她兴奋的是另一组数据。

热红外光谱——对准永久阴影区——显示了三微米附近的宽吸收带。水。

含水矿物，或者直接是水冰——在永久阴影坑底部，温度常年低于零下一百五十度，水冰可以稳定存在。

含量估算——初步的，不精确——阴影区表面含水量约百分之三到百分之五（质量分数）。

百分之三到百分之五。如果盘古总质量约十二亿吨，即使只有百分之一的质量是可提取的水冰——那也是一千两百万吨水。远航号五十人一年的用水量约七十三吨。一千两百万吨够用十六万年。

碳质物也有。C型特征的吸收带——三点四微米——出现在部分区域的表面光谱中。含量不高，可能是早期撞击混入的C型小行星碎片。但有碳就够了。碳加上铁，可以炼钢。碳加上氢（来自水），可以制造甲烷——化学推进的燃料。碳纤维复合材料的原料也有了。

苏晚调出一张汇总表。

| 成分 | 估算含量 | 估算总量 |
|---|---|---|
| 铁 | ~89% | ~10.7亿吨 |
| 镍 | ~8.5% | ~1.0亿吨 |
| 钴 | ~0.6% | ~720万吨 |
| 水冰 | ~1-3% | ~1200-3600万吨 |
| 碳质物 | ~0.2% | ~240万吨 |

她把表格发到全船通信频道。没有配文字。数字自己会说话。

三分钟后，赵阳在频道里说了一句："够了。"

苏晚没有回复赵阳的评论。她站在观察窗前，看着盘古的表面在两公里外缓缓旋转。大撞击坑的内壁上，暴露的金属断面在阳光下闪烁——那些银色的光斑是纯净的镍铁合金面，被四十六亿年前的某次撞击暴露出来，从此保持着金属的原始光泽。没有氧气，不会锈蚀。没有风雨，不会磨蚀。它在这里等了太阳系的全部历史。

她想起二十六年前，在天工一号的微重力实验舱里，她第一次用聚光镜熔化一块带上来的金属样品——那块样品重五十克，比一颗鸡蛋还轻。她用它验证了太阳能聚光熔炼在微重力下的可行性。那一天，她在笔记本上写了一行字："原理验证完成。"

现在，窗外是十二亿吨同样的金属。

她打开通信终端，开始写一条发给天工城的消息。通信延迟单程约十九分钟。等林远看到这条消息的时候，盘古又转了半圈。

消息很短：

"已抵达。盘古，M型，镍铁含量确认。有水。有碳。这一颗够用一百年。"

她按下发送。信号以光速离开远航号的通信天线，穿过二点九亿公里的真空。十九分钟后，它会到达天工城。

窗外，盘古继续旋转。银色的金属表面在阳光下一明一暗。像是一座沉默的矿山，在太空中等了四十六亿年，终于等来了第一批矿工。
