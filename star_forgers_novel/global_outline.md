# 《铸星者》 (The Star Forgers) - 全局大纲 v4

**目标篇幅**：100万字+ (约500章)
**核心母题**：在太空中建立自给自足的工业体系，最终在小行星带建立工厂——人类从行星文明走向星际文明的工程史。
**风格基调**：纯硬科幻、工程叙事、国家战略视角。

## 技术设定基石（全书一致，不得变动）

### 运输成本
- 背景：SpaceX Starship于2025年实现完全复用，LEO运输成本降至$200/kg。美国开始大规模太空商业化——Starlink二代、轨道数据中心、太空旅游。这刺激了中国的战略警觉。
- **长征九号甲**（可复用构型）：2028年首飞，2030年常态化运营。LEO载荷100-150吨。**成本约$500/kg**（初期），2035年降至$350/kg。
- 发射一座100吨模块 = ¥3.5亿。对国家级工程完全可承受。
- 中国不使用Starship。天工计划完全依赖国产运载。

### 太空太阳能（按太空场景设计，不是地面那套）
太空中的太阳能和地球完全不同——**更简单、更粗暴、更强大**：
1. **不需要光伏板**（至少不是主要手段）。太空中最高效的太阳能利用方式是**反射镜聚光**——一面薄如蝉翼的铝膜镜面（可以用小行星铝制造），在零重力下展开，把阳光聚焦到一个点上。不需要光电转换，直接用热。
2. **工业热源**：聚光镜将阳光聚焦到焦点，温度可达2000-3000℃。直接熔炼小行星矿石。不需要电炉，不需要燃料——阳光就是炉火。
3. **发电**：需要电力时，用聚光加热工质（如液态钠），驱动涡轮发电。或者用薄膜光伏（极轻，直接沉积在金属箔上）。
4. **零重力优势**：地面上的太阳能需要钢铁支架对抗重力、需要玻璃保护面板、需要清洁灰尘、需要储能应对夜晚。太空中这些全不需要。一张几微米厚的铝膜就是一面镜子，展开面积可以做到几平方公里，质量只有几十吨。
5. **近地轨道**：太阳常数1361W/m²，一面1km²的聚光镜 = 1.36GW热功率，相当于一座大型核电站。
6. **小行星带**（约2.5AU）：太阳常数降至~218W/m²（地球的1/6.25）。但在零重力下建造大面积镜面几乎没有成本，用6倍面积补偿即可。一面6km²的薄膜镜在小行星带仍可提供1.3GW热功率。

### 小行星资源
- 近地小行星（NEA）：距离近，适合早期捕获。但数量有限。
- **主小行星带**（2.1-3.3AU）：数百万颗小行星，资源几乎无限。M型小行星富含镍铁合金，C型富含水冰和碳化合物，S型含硅酸盐。
- 终极目标不是"把小行星拖回来"，而是**去小行星带建工厂**。

### 为什么人要跟工厂一起上天
远程操控延迟：LEO 0.5秒，月球2.5秒，小行星带16-32分钟（单程）。复杂制造不可能远程操控。人必须在现场。长征九号甲让送人成本可接受（每人约¥1500万，含训练和保障）。从第一天起就是有人值守。

### 生命保障
不是难题。CO₂通过Sabatier反应和水电解回收氧气。小行星水冰提供补充。全书不将"缺氧"作为矛盾。

## 故事起因：为什么是国家行为？

**2025-2026年**：SpaceX用Starship实现了月球货运、开始部署轨道数据中心，马斯克宣布2030年前送人去火星。美国太空商业生态爆发式增长。

中国航天界震动。不是恐惧，是**战略清醒**——太空正在变成新的工业疆域。谁先在太空中建立制造能力，谁就掌握了下一个世纪的战略主动权。这和当年"两弹一星"的逻辑一样：不是要不要做的问题，是不做就要落后的问题。

关键认知转变：过去中国航天的思路是"探索"——发射卫星、登月、探火。但现在，问题变成了**"生产"**——能不能在太空中造东西？能不能用太空的资源造？能不能在太空自给自足？

国务院批准**"天工"专项工程**：
1. **第一阶段（2028-2035）**：近地轨道有人值守制造平台，验证微重力制造和太阳能聚光熔炼。
2. **第二阶段（2035-2045）**：捕获近地小行星，实现太空原材料自给，完成自我复制验证。
3. **第三阶段（2045-2060）**：前进到主小行星带，建立永久性太空工业基地。

这不是某个企业家的冒险，是新时代的"两弹一星"。

## 发展路径（严格递进）

```
长征九号甲送人+设备到LEO
→ 有人值守制造平台"天工一号"（~50人）
→ 验证微重力制造（晶体、合金）
→ 部署第一面太阳能聚光镜（薄膜铝镜，聚焦熔炼测试）
→ 发射离子推进器去捕获一颗近地小行星（10-30米级M型）
→ 小行星拖至高轨，用聚光镜熔炼，提取镍铁和水冰
→ 用小行星金属制造更多聚光镜和结构件（自我复制起步）
→ 扩建为"天工二号"（~200人，大部分用太空材料建造）
→ 连续捕获多颗近地小行星，工业能力指数增长
→ 建造旋转重力居住区，千人级社区
→ 建造深空飞船，载人前往主小行星带
→ 在小行星带建立"天工三号"——永久性太空工业城市
→ 小行星带工业爆发：无限资源+太阳能=无限制造能力
→ 小行星带工业城市成熟，太空社会形成
```

## 第一卷：引力之井 [2025-2032]
**核心矛盾**：从零开始——在LEO建工厂，所有东西都从地球运，每一克都是钱。
**主线**：天工计划启动，建立第一座有人值守的太空制造平台。

**关键事件**：
- 开篇：SpaceX的爆发式发展。中国航天界的紧迫感。赵长河向高层汇报："美国人不只是在探索太空，他们在占领太空。"
- 天工计划立项。林远（航天结构工程师）被选为首批驻站人员。
- 长征九号甲首飞成功。天工一号模块逐批发射。约一年时间完成组装。
- 50名工程师进驻。他们不是宇航员式的"飞行访问"，而是真正的"太空工人"——住在那里，干活在那里。
- 苏晚主导微重力制造实验：完美单晶硅生长、金属合金凝固实验。
- 部署第一面太阳能聚光镜：直径200米的薄膜铝镜，在零重力下像花一样展开。聚焦阳光熔化测试金属样品。
- 工程困难：散热（太空中没有空气对流，只能辐射散热）、微陨石撞击镜面、设备在真空和温差中的退化。
- 人的困难：长期封闭环境、微重力身体退化、与家人分离、心理压力。
- 远程操控机器人去焊接一个结构件——信号延迟导致焊歪了。林远穿舱外服自己出去焊。"人得在现场。"
- 结尾：天工一号具备基本制造能力，但原材料100%来自地球。每公斤$500。必须找到太空中的资源。

## 第二卷：捕星者 [2032-2042]
**核心矛盾**：去哪里找资源？怎么把一块几千吨的太空石头拖回来？
**主线**：第一次近地小行星捕获，建立太空冶金能力，实现自我复制的第一步。

**关键事件**：
- 选定目标：近地小行星2031 TK₇，直径约18米，M型（镍铁），轨道参数适合用离子推进改变。估算质量约8000吨。
- "牧星一号"探测/推进器发射。太阳能离子推进，持续低推力改变小行星轨道。旅途约2.5年。
- 等待期间：天工一号升级聚光镜系统——从200米级扩大到500米级。更大的镜面=更高的聚焦温度=能熔炼真正的矿石。
- 驻站人口增至100人。建立更完善的生活区。第一次在太空中种菜（水培/气培）。
- 2034年底：2031 TK₇抵达高地球轨道。人类第一次拥有一座"太空矿山"。
- 熔炼：聚光镜聚焦阳光到小行星表面，温度超过1500℃。熔化的金属在微重力下形成球状液滴，用磁场引导收集。镍铁合金分离。
- 水冰提取：小行星含少量水冰，加热蒸发后收集。电解为氧气和氢。
- **关键里程碑**：用小行星镍铁轧制出金属板材，用它制造了一面新的聚光镜支架。太空工厂第一次用太空材料扩展了自身。
- 国际反应：美国加速自己的太空制造计划。欧洲、日本提出合作意向。
- 结尾：自我复制的原理已验证。但近地小行星数量有限，且多为小型。真正无限的资源在主小行星带——2.5AU之外。

## 第三卷：锻造之环 [2042-2055]
**核心矛盾**：从"能造"到"自己长"——太空工业的指数增长，以及向小行星带的战略推进。
**主线**：连续捕获近地小行星实现指数扩张，建造旋转居住区，建造深空飞船前往小行星带。

**关键事件**：
- 连续捕获近地小行星：每次捕获的小行星提供材料来制造更多的推进器、更多的聚光镜、更多的加工设施。2年捕一颗 → 1年两颗 → 半年三颗。指数增长。
- 制造能力暴涨：天工基地从100人扩到500人，大部分新增结构用太空金属建造。
- **旋转居住区建造**：第一个O'Neill圆柱体雏形——直径200米、长500米的旋转舱段。0.3g人工重力。人可以在里面"走路"。第一代太空儿童在这里出生。
- 太空农业：旋转居住区内建立大规模水培农场。用聚光镜引入自然光种植。粮食自给率从10%爬升至70%。
- 太空工业反哺地球：在太空中制造的大型通信卫星、太空望远镜镜片——这些东西在太空造比从地球发射更便宜。
- **深空飞船设计**：要去小行星带（约2.5AU），飞行时间约1.5-3年（取决于推进方式）。需要一艘大型载人飞船——在太空中建造，不需要承受大气和重力，可以做到地面不可能的尺寸。
- "远航号"建造：全长300米，太阳能电推+化学推进混合。载员50人。全部用太空金属在轨道上建造。
- 远航号出发，目标：主小行星带中一颗直径约800米的M型小行星"盘古"。
- 新一代成长：林远的女儿林小萱在太空长大，成为远航号的工程师之一。
- 结尾：远航号抵达"盘古"小行星。站在一座800米的金属山上，他们看到了无限的可能。

## 第四卷：带区工厂 [2055-2070]
**核心矛盾**：在小行星带建立自给自足的工业城市——距地球16-32分钟通信延迟，必须完全自主。
**主线**：在主小行星带建立"天工三号"，从前哨站发展为自给自足的工业基地。

**关键事件**：
- "盘古"小行星的开发：部署大型聚光镜阵列。太阳光在2.5AU弱了6倍，但薄膜镜面积做到数平方公里补偿。
- 用"盘古"的金属建造"天工三号"——围绕小行星建设的太空工厂+居住区。从50人起步。
- 自给自足的考验：通信延迟16-32分钟（单程），天工三号必须在供应链、决策、医疗等方面完全自主。
- 人口增长：更多工程师从地球出发，旅途约2年。天工三号人口从50增至500。
- 工业能力扩展：开始向附近小行星派出采矿船。
- 旋转居住区扩建——从200米级扩到500米级。内部有农场、医疗、教育设施。
- 与地球的关系变化：天工三号越来越独立，但人才和信息纽带仍在。不是"独立"，是"分支"。
- 林远（60多岁）在天工三号安顿下来——他参与建造了从天工一号到天工三号的所有设施。
- 结尾：天工三号实现完全自给自足。从小行星带望回去，地球只是一个亮点。但他们建造的一切，都是为了让那个亮点上的人过得更好。

## 第五卷：新世界 [2070-2085]
**核心矛盾**：太空工业体系已经自给自足——接下来做什么？如何定义太空中诞生的新社会？
**主线**：小行星带工业城市的成熟，太空社会的形成，以及人类文明格局的重塑。

**关键事件**：
- 天工三号工业能力持续增长：从一座工厂变成一片工业区——多颗小行星被开发，形成"天工星群"。
- 大型O'Neill圆柱体建造：直径1公里、长3公里。内部有人造天空、公园、河流。常驻人口数千人。
- 第二代太空人（林小萱这代）成为主力。他们在太空出生长大，对"太空生活"没有恐惧和新鲜感——这就是他们的日常。
- 太空工业反哺地球进入成熟期：大型基础设施组件、稀有金属精炼品、特种材料从小行星带运回近地轨道。
- 太空社会治理问题浮现：数千人的社区不再是"工程队"，而是一个社会。需要制度、文化、身份认同。
- 国际格局变化：其他国家也在小行星带建立了自己的基地。合作与竞争并存。
- 木星系无人探测：从小行星带出发更近。探测器抵达木卫二，发现疑似微生物的化学痕迹——留下开放性悬念。
- 林远（70多岁）在天工城安详退休。他从窗口看出去，看到自己34岁时焊的第一根梁，现在已经是整座城市的一部分。
- 结尾：林小萱站在O'Neill圆柱体内部，头顶是人造的蓝天，脚下是小行星金属铸成的大地。远处，新一批从地球来的年轻工程师正在出舱。一切都在继续。

- 林远的女儿。在太空中出生和长大。第二代太空人的代表。远航号工程师，后成为天工三号的核心技术负责人。

## 全书主题
- 太空工业是工程问题，不是幻想。每一步有计算、有成本、有失败。
- 聚光镜+小行星=自我复制的工业体系。这是指数增长的关键。
- 国家意志+工程师精神，是人类突破引力井的方式。
- 小行星带才是人类太空工业的真正舞台——近乎无限的资源，足够的阳光，零重力下的自由制造。
- 故事止步于太空工业体系的建成和太空社会的诞生。不急着飞出太阳系——把脚下的事做扎实。
